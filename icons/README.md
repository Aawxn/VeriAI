# Icons Directory

This directory should contain the extension icons in the following sizes:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon32.png` - 32x32 pixels (Windows)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

For development, you can use placeholder icons or create simple icons using any image editor.

The icons should represent the AI Ethics Monitor concept - perhaps a combination of:
- AI/brain symbol
- Ethics/balance symbol
- Monitor/eye symbol
- Shield for protection/safety

## Temporary Development Icons

For development purposes, you can create simple colored squares or use online icon generators until proper icons are designed.
